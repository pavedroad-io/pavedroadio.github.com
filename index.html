What is a Paved Road?

The term “Paved Road” was coined by the Netflix tools teams which created several fully integrated end-to-end toolchains for writing, testing, deploying, and operating their streaming video service. For each support execution framework such as Java, Python, or Go, an integrated working CI/CD toolchain was created. This method of pre-integrated and tested toolchains converts a bumpy and challenging road into a delightfully smooth road which dramatically increases the velocity of development teams.

Designed for Developers

Quickly enable a cloud-native development environment (IDE, development toolchains), CI/CD, microservices, operations, and GitHub repositories that are auto-populated with everything you need to pursue cloud-native development.

Time Saver

No need to spend months investigating cloud-native toolchains for development, deployment, and operations. Adopt the already curated, integrated, and tested open source PavedRoad toolchains, whether you are developing a greenfield cloud-native project, or adapting an existing application to the advantages of cloud-native.

Easy to Customize and Extend

As an open source project, any community developer can extend PavedRoad.

Cloud-Native Focused

Focused 100% on cloud-native tools such as containers, Kubernetes, service meshes, CI/CD pipelines, microservices, and serverless functions.

Features

  • Cloud-Native multicloud integrated toolchain for development, CI/CD, and operations
  • Support for the most popular cloud-native IDE's and languages
  • Automated GitHub repository creation based on profile: OSS, enterpise, or individual projects
  • Template-based file generation based on repository profile
  • Container compatible CI/CD
  • Advanced analytics across the entire delivery chain for your organization or comparative data looking at other customers using the same toolchain
  • 100% Open Source
