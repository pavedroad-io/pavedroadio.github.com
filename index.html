What is a Paved Road?

The term “Paved Road” was coined by the Netflix tools teams which created several fully integrated end-to-end toolchains for writing, testing, deploying, and operating their streaming video service. For each support execution framework such as Java, Python, or Go, an integrated working CI/CD toolchain was created. This method of pre-integrated and tested toolchains converts a bumpy and difficult road into a delightfully smooth road which dramatically increases the velocity of development teams.

Designed for developers

Quickly enable a cloud-native development environment (IDE, development tool chains),CI/CD, MicroServices, Operationas, and GitHub repositories that are auto populated with everything you need to pursue cloud native development.

Time saver

No need to spend months investigating cloud-native tool chains for development, deployment, and operations. Simply adopt the already curated, integrated and tested Open Source PavedRoad tool chains, whether you are developing a green field cloud-native project, or adapting an existing application to the advantages of cloud-native.

Easy to customise and extend

As an Open Source Project, any community developer can extend PavedRoad.

Cloud-Native focused

100% Focused on Cloud Native tools such as Containers, Kubernetes, Service Meshes, CI/CD pipelines, Microservices, and Serverless Functions

Features

  • Cloud-Native multicloud integrated tool chain for development, CI/CD, and Operations
  • Support for the most popular Cloud Native IDE's and languages
  • Automated GitHub repository creation based on profile: OSS, Enterpise, or Individual projects
  • Template based file generation based on repository profile
  • Container compatible CI/CD
  • Advanced analytics across the entire delivery chain for your organization or comparative data looking at other customers using the same toolchain
  • 100% Open Source
